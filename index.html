Tutoriais basicos para o uso de Smartphone

Passo a passo para a utilização das funcionalidades principais

Como fazer uma chamada telefônica:

1. Abra o aplicativo Telefone.
2. Digite o número de telefone desejado.
3. Toque no botão Ligar.





Como enviar uma mensagem de texto:

1. Abra o aplicativo Mensagens.
2. Toque no botão Nova Mensagem.
3. Digite o número de telefone do destinatário.
4. Escreva sua mensagem e toque em Enviar.
